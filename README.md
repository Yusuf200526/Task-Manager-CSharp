# To-Do List 📋

A To-Do List application that helps you set your tasks, customize them, mark them with flags based on priority, and save them to files — built with **C#** and **Windows Forms (.NET Framework)**, developed to apply the concepts learned in Dr. Mohammad Abu-Hadhoud's C# Level 1 course on [Programming Advices](https://www.programmingadvices.com).

## Features

- Add tasks with a name and priority level (Urgent / Medium / Low)
- Priority flags displayed as colored icons next to each task
- Timestamp recorded automatically when a task is created
- Mark tasks as achieved using the checkbox
- Filter tasks by: All / Achieved / Priority (Urgent, Mid, Low)
- Right-click any task to Delete or Edit it
- Delete all tasks at once with the trash button
- Tasks saved to files so they persist between sessions
- Quick access to the author's GitHub profile via the GitHub button
- Friendly empty-state illustrations when no tasks are present

---

## How to Use

1. Click the **+** button to add a new task
2. Enter the task name and select its priority level
3. Click the confirm button to save the task
4. Check the checkbox next to a task to mark it as achieved
5. Right-click any task to **Edit** or **Delete** it
6. Use the **View** menu to filter tasks by status or priority
7. Click the trash icon to delete all tasks at once
